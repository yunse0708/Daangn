## 💡 왜 당근마켓인가?

또한, 전부터 클론코딩을 해보고 싶다는 생각이 있었습니다. <br>
그래서 이번 기회를 통해 한 번 해보자라는 계기로 시작하게 되었습니다.

      - 중간에 코드가 조금 날라가서 styled-components가 기존과 달라 화면에 보여지는 것도 달라져 그 부분이 아쉬웠습니다. <br>
